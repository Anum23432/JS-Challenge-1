Logical And Assignment Operators.

Assignment operators in which we assign the value to variables.
If = signs comes with any arithmetic operator it will give a new value to
the variable and that will be assignment operator.
Logical operators are of three types;

  • And && operators(if both conditions are true the function will be performed).
  • Or || operators (if only one condition will be true the function is executed.)
  • NOT ! operators(it will give the opposite output of the condition.)


So , through this function as we have changed the conditions and checked function is working or not.
If we change the logical operator between them it will work according to its specificity.
And in not operator the function is working in totally opposite.
